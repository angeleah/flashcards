Flashcards
==========
This is a web app to quiz yourself on the Ruby core methods. Select a Class or Module and quiz yourself.

Usage:
------
- Getting started:
  + clone the app.
  + cd into the directory.
  + using ruby 2.0.0-p353
  + bundle.
  + rake db:migrate
  + rake db:test:prepare
  + rake db:seed

- To run the app:
  + start the server.
  + navigate to localhost:3000

- To run the specs:
  + rspec
